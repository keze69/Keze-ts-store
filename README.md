# Keze TS Store

This is a demo single-page clothing store made for direct upload to **GitHub Pages**.
- Theme: Modern dark with colorful neon accents
- 9 example products with SVG placeholder images
- Demo "Pay with GPay" button (not a real payment integration)
- Mobile-friendly and ready to upload

## To publish on GitHub Pages
1. Create a repository (public).
2. Upload these files to the repository root.
3. In **Settings → Pages**, set Source to **main** branch and folder **/root**.
4. Wait a few moments and open `https://<yourusername>.github.io/<repo-name>/`

Enjoy!
